<wc-bell-ring/>

A simple ringing bell component.

Demo here: https://codepen.io/annoyingmouse/pen/gOyQeew

Installation

<script type="module"
        src="https://unpkg.com/@annoyingmouse/wc-bell-ringing/wc-bell-ring.js"></script>

Alternatively:

<script type="module"
        src="https://cdn.skypack.dev/@annoyingmouse/wc-bell-ringing/wc-bell-ring.js"></script>

You can use it minified by adding using this instead:

<script type="module"
        src="[WHATEVER CDN]/@annoyingmouse/wc-bell-ringing/wc-bell-ring.js"></script>

Please remember to change [WHATEVER CDN] to your favourite CDN

Configuration

  • Adding a times attribute will dictate the number of times the bell will ring (hint, you can set it to "infinite").
  • Adding a no-shadow attribute and setting it to true will hide the subtle drop shadow.
  • Adding a font-size attribute with a valid CSS value will allow you to alter the size of the bell.
  • Adding a click-to-ring attribute and setting it to true will allow users to click the element to ring it.
  • Adding a duration attribute with a valid CSS value will alter the speed of the ringing.
